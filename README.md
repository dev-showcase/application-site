# Application-Site

## About this Project

This is my personal application website, designed to present myself as a developer and showcase selected projects, technical skills, and relevant experience for potential employers or collaborators.

The site is fully responsive, fast, and built with modern web technologies. It serves as both an online CV and a portfolio, tailored for job applications in the tech industry.

### Features
- Clean and responsive design
- Project showcase
- Short bio, skills overview and cv
- Contact section

### Purpose
To provide a professional, developer-focused introduction that highlights my capabilities and projects, beyond what a traditional CV can offer.
